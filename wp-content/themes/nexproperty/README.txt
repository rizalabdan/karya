Real Estate Directory Theme for Real Estate Agency, but will help you to run any type of Directory Business.

== Description ==

Listing Directory theme for Real Estate Agency, includes also Car Dealer demo content but will help you to run any type of Directory Business.

Theme is mostly build for Elementor so everything can be customized visually.

<h3>Real Estate Agency Directory Features</h3>
<ul>
 	<li>Manage Listings</li>
	<li>Manage Categories</li>
	<li>Manage Fields</li>
	<li>Select listings agent</li>
	<li>Manage indoor amenities</li>
	<li>Manage outdoor amenities</li>
	<li>Multimedia fields integration</li>
    <li>Full Elementor compatibility</li>
    <li>Messaging support</li>
    <li>Demo data import for Real Estate and Car Dealership</li>
    <li>Open street maps support</li>
</ul>

<h3>Theme using following free plugins to accomplish functionality:</h3>

<ul>
    <li><a href="https://wpdirectorykit.com/">WpDirectoryKit</a></li>
    <li><a href="https://elementor.com/">Elementor</a></li>
    <li><a href="https://elementinvader.com/">ElementInvader</a></li>
    <li><a href="https://elementinvader.com/">ElementInvader Addons</a></li>
    <li><a href="https://ocdi.com/">One Click Demo Import</a></li>
</ul>

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Theme works perfectly and by default use Elementor, WpDirectoryKit, ElementInvader, ElementInvader Addons
